أداة ويب آمنة ومفتوحة المصدر لتوليد كود جلسة الحساب الشخصي بخصوصية تامة.

الخطوة 1: بيانات المطور ورقم الهاتف
يرجى إدخال الـ API ID الخاص بك.
يرجى إدخال الـ API HASH الخاص بك.
يرجى إدخال رقم الهاتف كاملاً مع رمز الدولة.

يرجى إدخال رمز التحقق المستلم.
تفقد رسائل حساب تليجرام الخاص بك للحصول على الكود.
يرجى كتابة كلمة مرور التحقق بخطوتين لحسابك.
تم التوليد بنجاح!
